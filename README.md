<h1 align="center">徐州市垃圾收运系统</h1>

> 基于高德地图API的徐州市垃圾分类收运系统的设计与实现

### 🏠 [Homepage](https://xuzhou-waste-system.netlify.com/)

## 环境要求

- node >=8.9
- npm >= 3.0.0

## 安装

```sh
npm i
```

## 运行

```sh
npm run dev
```
